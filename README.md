# BetterGenZ
Minecraft plugin for Spigot 1.12.2!

Author:
- damianoR

This plugin adds Generators to SuperiorSkyblock servers!

With a generator, you can spawn multiple ores:
- coal
- iron
- gold
- diamond
- emerald
- netherite

There are 3 different generators, from most simple to very smart with chest and production rate multiplied:
- Base
- Smart
- Smart++
